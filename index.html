DEVON

Un asistente virtual con IA

¿Qué es DEVON?

Usando la tecnología de GPT4All y el modelo de lenguaje natural LLM Nous Hermes 2 Mistral 7B, podemos lograr un asistente capaz de resolver dudas algo más avanzadas y poco convencionales. Aunque todavía está en desarrollo y tiene algunos fallos e incongruencias, DEVON ofrece una interfaz agradable y está en constante mejora.

Integraciones necesarias

Para hacer funcionar DEVON necesitarás obtener las siguientes APIs:

  • Spotify API
  • News API
  • OpenWeatherMap API
  • RapidAPI

Además, DEVON requiere Vulkan SDK y CUDA para funcionar correctamente.
